  • Tag List
em
Emphasis, an updated version of the italics tag with improved accessibility.
strong
Like em, an updated version of the bold tag.
img
Adds an image to the page. The src and alt attributes are always needed, in order to attach the image and provide accesibility.
h1 to h6
Used for headings. Should be used in order of importance, with h1 being used for things like page titles.
title
Determines the page's title, which appears in the browser's tabs. Will not add anything to page body.
